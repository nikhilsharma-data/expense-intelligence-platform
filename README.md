# 💰 Expense Intelligence Platform

A full-stack data application to track, analyze, and visualize personal expenses with actionable insights.

---

## 🚀 Features

- 📊 Dashboard with KPIs (Income, Expense, Balance)
- 📈 Category-wise expense breakdown
- 🍩 Expense distribution (Top categories + Others)
- 📉 Monthly cashflow trend
- 🧠 Smart insights (spending patterns, warnings)
- 📅 Date filters (30d, 90d, YTD)
- 📂 CSV upload support

---

## 🛠 Tech Stack

- **Frontend**: Streamlit
- **Backend**: FastAPI
- **Database**: PostgreSQL
- **Language**: Python
- **Visualization**: Matplotlib
